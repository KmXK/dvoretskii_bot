### Профиль человека

/me - команда которая позволяет человеку просмотреть свой профиль, вывод состоит из блоков и кнопок

Блоки:
- достижения - в формате строки состоящей из иконок достижений

Кнопки:
- Достижения - "подробнее о достижениях"

Подробнее о достижениях - отправляется отдельное сообщение где будут все достижения человека с пагинацией в формате
id. медалька - наименование

### Достижения

/rewards - посмотреть общий список достижений с пагинацией в формате
id. медалька - наименование

... add - добавить новое достижение
бот спрашивает наименование/описание
пользователь отвечает
бот спрашивает медальку
пользователь отправляем смайлик (смайлики могут быть из кастомных телеграмм наборов)

... remove {id} - удалить достижение, удаляет так же у тех кому оно присвоено

... {id медали} present {id пользователя/username} {id пользователя/username} ...   - вручить медаль

... {id медали} take {id пользователя/username} {id пользователя/username} ... - изъять медаль
