### Профиль человека

/me - команда которая позволяет человеку просмотреть свой профиль, вывод состоит из блоков и кнопок

Блоки:
- достижения - в формате строки состоящей из иконок достижений

Кнопки:
- Достижения - переключает текущее сообщение на список достижений с пагинацией

Подробнее о достижениях - текущее сообщение редактируется и показывает список достижений с пагинацией в формате
id. медалька - наименование

Есть кнопка "← Назад" которая возвращает к общему профилю

### Достижения

/rewards - посмотреть общий список достижений с пагинацией в формате
id. медалька - наименование

... add - добавить новое достижение
бот спрашивает наименование/описание
пользователь отвечает
бот спрашивает медальку
пользователь отправляем смайлик (смайлики могут быть из кастомных телеграмм наборов)

... remove {id} - удалить достижение, удаляет так же у тех кому оно присвоено

... {id медали} present {id пользователя/username} {id пользователя/username} ...   - вручить медаль

... {id медали} take {id пользователя/username} {id пользователя/username} ... - изъять медаль

### Список событий

для каждого чата свои события
очень похоже на fq 

/todo - показать список событий для этого чата с пагинацией

/todo {текст} - добавить событие

... remove {id} - удалить событие

... done {id} - завершить событие
бот завершает событие
бот спрашивает не хотите ли вы добавить достижение (кнопки пропадают после нажатия)
если нет то все
если да то бот спрашивает про название достижения, медаль и кому присвоить(можно нескольким людям)
